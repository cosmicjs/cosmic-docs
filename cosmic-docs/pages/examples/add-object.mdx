# Add Object

Use the Cosmic API to add Objects to your Bucket. For more information on adding Objects, go to the [add Object section](/api-reference/objects#add-object) in the docs.

### Setup

**Step 1.** Log in to your [Cosmic dashboard](https://app.cosmicjs.com/login) and go to <i>Bucket Settings > API Accesss</i> to get your Bucket `slug`, `read_key`, and `write_key`.

**Step 2.** Create an Object Type for "Products" with the following metafields: Price (`Number`) and Additional Info (`Textarea`).
