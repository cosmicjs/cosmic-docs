# Basic

We'll be using the [Simple React Blog](https://www.cosmicjs.com/apps/simple-react-blog) app template for the following basic query examples. You can install it then switch out the `BUCKET_SLUG` and `BUCKET_READ_KEY` in the following examples to your Bucket. Find these values located in <i>Your Bucket > Settings > API Access</i> after logging in to your <a href="https://app.cosmicjs.com">Cosmic dashboard</a>.

## Get Objects by Type

The following example shows you how to perform a basic query to get multiple Objects from your Cosmic Bucket using the `type` query parameter. Copy and paste this into your code editor or terminal to see actual data returned.

**Endpoint**

```bash
GET /v2/buckets/${YOUR_BUCKET_SLUG}/objects
```

**Query**

```json
{
  "type": "posts"
}
```

<Tabs>
<div label="Node.js">
```javascript
const api = require('cosmicjs')()
const bucket = api.bucket({
  slug: 'simple-react-blog',
  read_key: '' // Find this in Bucket > Settings > API Access after logging in https://app.cosmicjs.com/login
})
const app = async () => {
  const data = await bucket.getObjects({
    query: {
      type: 'posts'
    },
    props: 'slug,title,metadata.hero.imgix_url,metadata.author.title'
  })
  const posts = data.objects
  console.log(posts)
}
app()
```

</div>
<div label="cURL">
```bash
curl https://api.cosmicjs.com/v2/buckets/simple-react-blog/objects \
	-d read_key="" \
	--data-urlencode query='{"type":"posts"}' \
	-d props="slug,title,metadata.hero.imgix_url,metadata.author.title" \
	-G
```
</div>

## Get Object by Slug

The following example shows you how to perform a basic query to get a single Object from your Cosmic Bucket using the `slug` query parameter. Copy and paste this into your code editor or terminal to see actual data returned.

**Endpoint**

```bash
GET /v2/buckets/${YOUR_BUCKET_SLUG}/objects
```

**Query**

```json
{
  "slug": "a-wonderful-blog-post-about-earth"
}
```

<Tabs>
<div label="Node.js">
```javascript
const api = require('cosmicjs')()
const bucket = api.bucket({
  slug: 'simple-react-blog',
  read_key: '' // Find this in Bucket > Settings > API Access after logging in https://app.cosmicjs.com/login
})
const app = async () => {
  const data = await bucket.getObjects({
    query: {
      slug: 'a-wonderful-blog-post-about-earth'
    },
    props: 'slug,title,metadata.hero.imgix_url,metadata.author.title'
  })
  const post = data.objects[0]
  console.log(post)
}
app()
```

</div>
<div label="cURL">
```bash
curl https://api.cosmicjs.com/v2/buckets/simple-react-blog/objects \
	-d read_key="" \
	--data-urlencode query='{"slug":"a-wonderful-blog-post-about-earth"}' \
	-d props="slug,title,metadata.hero.imgix_url,metadata.author.title" \
	-d limit=1 \
	-G
```
</div>
