### Encode queries

If you are using [Queries](/api-reference/queries) in Postman here's a handy way to encode your `query` param:

1. Add your query as a `param` in your Postman request and select the input or right click:
   ![Encode Query Screenshot 1](https://cdn.cosmicjs.com/b43bf5e0-675d-11eb-9ea4-1be290e7bb91-query-1.png)

2. Click the three dots and select "Encode URI Component":
   ![Encode Query Screenshot 2](https://cdn.cosmicjs.com/b43f2a30-675d-11eb-9ea4-1be290e7bb91-query-2.png)

3. See your query now encoded and ready to use:
   ![Encode Query Screenshot 3](https://cdn.cosmicjs.com/b4414d10-675d-11eb-9ea4-1be290e7bb91-query-3.png)
