# Introduction

Welcome to the Cosmic v2 REST API docs. This is currently in an `alpha` version and your feedback is welcome and appreciated. See [the changelog](/cosmic-api/changelog) for details on what has changed in v2.

For the stable v1 release of the REST API, go to [https://docs.cosmicjs.com](https://docs.cosmicjs.com).

## Base URL

To keep things concise, all requests use the `$BASE_URL` variable to indicate the following base URL:

```bash
https://api.cosmicjs.com/v2
```
