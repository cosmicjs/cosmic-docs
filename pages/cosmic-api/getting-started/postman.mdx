# Postman

Review and interact with REST API endpoints using Postman.

## Getting started

1. Click the button below to install the collection into your Postman account:

   <a href="https://app.getpostman.com/run-collection/799f70a575df19a27f8b">
     <img
       src="https://run.pstmn.io/button.svg"
       alt="Run in Postman"
       className="w-1/5"
     />
   </a>

2. Click the dots then "Edit" option to get into the "Cosmic API V2" collection settings

   <img
     alt="Postman settings"
     src="https://cdn.cosmicjs.com/7b777500-6018-11eb-ad8f-d1a7eae548e8-postman-1.png"
     className="w-6/12"
   />

3. Edit your variables to match with your Cosmic user account and Bucket API access keys.
   <img
     alt="Postman varialbes"
     src="https://cdn.cosmicjs.com/7b857ec0-6018-11eb-ad8f-d1a7eae548e8-postman-2.png"
     className="w-full"
   />
