# Introduction

Welcome to the Cosmic REST API v2 docs. This is currently an `alpha` release and your feedback is welcome and appreciated. See [the changelog](/api-reference/changelog) for details on what has changed in v2.

For the stable v1 release of the REST API, and interactions with the Cosmic NPM module and GraphQL API, go to [https://docs.cosmicjs.com](https://docs.cosmicjs.com).

## REST Base URL

To keep things concise, all requests to the REST API use the `$BASE_URL` variable to indicate the following base URL:

```bash
https://api.cosmicjs.com/v2
```

## Node.js

Install the official NPM module for Cosmic with the following command:

```bash
npm i cosmicjs
```

## GraphQL Base URL

All GraphQL API examples use the following API endpoint:

```bash
https://graphql.cosmicjs.com/v2
```

## Status Check

You can check on the API system status via the [Cosmic status page](https://cosmicjs.statuspage.io/) and via the following endpoint:

```
https://api.cosmicjs.com/v2/status
```

## Postman

We offer a [Postman](https://www.postman.com/) collection for a quick reference to review and interact with all Cosmic REST API endpoints. Follow the steps to install and configure below.

### Install and configure

1. Click the button below to install the collection into your Postman account:

   <a href="https://app.getpostman.com/run-collection/d18f51536ac067f0e4d1">
     <img
       src="https://run.pstmn.io/button.svg"
       alt="Run in Postman"
       className="w-1/5"
     />
   </a>

2. Click the dots then "Edit" option to get into the "Cosmic API V2" collection settings

   <img
     alt="Postman settings"
     src="https://imgix.cosmicjs.com/d54cdae0-68c2-11eb-82fa-03c9fae183bd-postman-1.png?auto=format&w=1000"
     className="w-6/12"
   />

3. Edit your variables to match with your Cosmic user account and Bucket API access keys.

   <img
     alt="Postman varialbes"
     src="https://imgix.cosmicjs.com/0bf7e940-68c3-11eb-82fa-03c9fae183bd-postman-2.png?auto=format&w=2000"

   ### Encode queries

   If you are using [Queries](/api-reference/queries) in Postman here's a handy way to encode your `query` param:

   2. Click the three dots and select "Encode URI Component":

   3. See your query now encoded and ready to use:
