# Introduction

Welcome to the new documentation site for the [Cosmic](https://www.cosmicjs.com) headless CMS. [What is a headless CMS](https://www.cosmicjs.com/headless-cms)?

## Overview

This is the docs site for the v2 release of the REST API (currently in alpha). For the stable v1 release of the REST API, go to [https://docs.cosmicjs.com](https://docs.cosmicjs.com).

Find references to READ, ADD, EDIT, and DELETE content from your Cosmic Bucket. All requests require an account on Cosmic. [Sign up for a free account](https://app.cosmicjs.com/signup).
