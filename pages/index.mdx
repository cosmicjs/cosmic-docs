# Introduction

Welcome to the new documentation site for [Cosmic](https://www.cosmicjs.com).

## What is Cosmic?

Cosmic is a [headless CMS](https://www.cosmicjs.com/headless-cms) which provides a web dashboard to create content and API tools to deliver content to any website or app.

## Overview

This is the docs site for our new [REST API v2](/cosmic-api/getting-started/introduction) (currently in an alpha release). For the stable v1 release of the REST API, go to [https://docs.cosmicjs.com](https://docs.cosmicjs.com).

This new docs site is build on Next.js and hosted on Vercel. Find references to READ, ADD, EDIT, and DELETE content from your Cosmic Bucket. All requests require an account on Cosmic. [Sign up for a free account](https://app.cosmicjs.com/signup).
