# Introduction

Welcome to the new documentation site for Cosmic. We are currently alpha testing our [REST API v2](/cosmic-api/getting-started/). For the stable v1 release of the REST API, go to [https://docs.cosmicjs.com](https://docs.cosmicjs.com).

## Overview

This new docs site is build on Next.js and hosted on Vercel. Find references to READ, ADD, EDIT, and DELETE content from your Cosmic Bucket. All requests require an account on Cosmic. [Sign up for a free account](https://app.cosmicjs.com/signup).
